Drag and Drop API

概要

・ draggable="true/false"でドラッグできるかどうかを決める。

・ 要素をマウスでクリックしてドラッグを開始 → dragstartイベントが発生

・ ドラッグ中、マウス下の要素が変わる → ドラッグ先でdragenterイベントが発生

・ ドラッグ中、その要素では、dragイベントが発生(※一定時間ごとに発生)

・ ドラッグ先の要素では、dragoverイベントが発生(※一定時間ごとに発生)

・ ドラッグが終わったときに、ドラッグ先でdropイベントが発生

・ ドラッグが終了したときは結果にかかわらず、ドラッグ元でdragendイベントが発生

【ドラッグ元】dragstart→drag→drag→……→drag→dragend

【ドラッグ先】dragenter→dragover→dragover→……→dragover→dragleave
